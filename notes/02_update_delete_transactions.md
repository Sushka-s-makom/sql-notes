# Урок 2 — update/delete и транзакции

update / delete без where затрагивает все строки.

Безопасный приём:
1) сначала выполнить select с тем же where и проверить, какие строки выберутся
2) затем выполнить update/delete

Транзакции:
- begin — начать транзакцию
- rollback — отменить всё, что сделано после begin
- commit — зафиксировать изменения навсегда
