# Урок 1 — первая таблица

Таблица хранит строки (записи) и колонки (поля).

SERIAL = целое поле + авто-нумерация через sequence.
PRIMARY KEY = уникальность + NOT NULL + индекс.
NULL = “значение неизвестно/не задано” (не равно 0 и не равно пустой строке).

Команды:
- CREATE TABLE — создать таблицу
- INSERT — вставить строки
- SELECT — выбрать
- WHERE — фильтр
- ORDER BY ... NULLS FIRST — сортировка и управление NULL
