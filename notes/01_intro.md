# Введение

PostgreSQL запущен в Docker.
Подключение выполняю через DBeaver.

Проверочные запросы:
- SELECT 1
- SELECT version()
- SELECT current_database()
- SELECT current_user()

Понял, что SQL — это язык запросов к базе данных,
а Postgres — программа, которая эти запросы выполняет, всё это внутри Docker.
